# autoresearch

本项目基于 [mhpi/generic_deltamodel](https://github.com/mhpi/generic_deltamodel) 开发，核心依赖其 Delta Model / dPL 相关能力。

## 说明

- 这是一个在 `generic_deltamodel` 基础上进行的研究与实验项目。
- 当前仓库主要围绕 **Delta Model（dPL）** 相关训练、评估与分析流程展开。
- 如果你之前只记得 “delta 模型”，可以直接理解为：这个项目是建立在 `generic_deltamodel` 提供的 Delta Model 框架之上的扩展实现。

## 相关位置

- 配置文件：`conf/`
- 核心实现：`implements/`
- 项目实验脚本：`project/Invariant_dpl/`
- 数据目录：`data/`
